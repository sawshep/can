# Usage

## Trash files

`can foo.txt bar.d`

## List files in trashcan

`can -l`

## List files in trashcan that match a regex

`can -l '^foo'`

## View trashinfo of files

`can -n foo.txt bar.d`

## Untrash files

`can -u foo.txt bar.d`

## Empty files from trashcan

`can -e foo.txt bar.d`

## Empty entire trashcan

`can -e`
